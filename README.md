# DF-Sim: 

installation : 

* `$ sudo apt install libboost-dev`
* `$ sudo apt install make` 
* `$ sudo apt install g++`


how to run :

* `$ ./raid [config_file_name] [trace_file_name]`

config_file_name => ssd.config | 
tarce_file_name  => /trace/trace.txt

example : `./raid ssd.conf /trace/trace0.txt`
